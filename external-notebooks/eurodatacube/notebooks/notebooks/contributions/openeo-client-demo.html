openEO Client Demo

This notebook demonstrates how an openEO job execution can be managed from Python.

To run this notebook, an openEO early adopters account is necessary. Please obtain one via this link: https://openeo.cloud/early-adopters/

import openeo
connection = openeo.connect('https://openeo.cloud').authenticate_oidc()
